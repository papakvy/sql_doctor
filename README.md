# SQL Doctor v1.0.0

## Pre-requirements

- `shc` package must be installed on your system

## Installation Instructions

### 1. Install

```bash
make install
```

Then, follow the messages during installation!

### 2. Verify

```bash
sql_doctor -v

# sql_doctor 1.0.0 (2023-11-22)
```

### 3. Uninstall

```bash
make uninstall
```
